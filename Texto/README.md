# Texto

Principales comandos utilizados en los scripts para **procesamiento y manipulación de texto** en Bash, según los trabajos prácticos realizados.

---

## Comandos principales

- **`cat`**  
  Muestra el contenido completo de archivos de texto o concatena varios archivos en uno solo.

- **`echo`**  
  Imprime texto o variables por consola o hacia archivos.  
  Se utiliza para mostrar mensajes, resultados o generar contenido dinámico.

- **`awk`**  
  Permite procesar texto por columnas o mediante patrones.  
  Es útil para filtrar, calcular y transformar datos en archivos estructurados (como tablas, logs o CSV).

- **`sed`**  
  Editor de flujo de texto. Se usa para modificar, eliminar o reemplazar texto dentro de archivos o líneas específicas.  
  También permite eliminar rangos de líneas o insertar contenido en posiciones determinadas.

- **`grep`**  
  Busca coincidencias de texto dentro de archivos o resultados de comandos.  
  Ideal para filtrar información o encontrar patrones específicos.

- **`wc`**  
  Cuenta líneas, palabras o caracteres de un archivo.  
  Muy útil para obtener estadísticas básicas de texto.

- **`cut`**  
  Extrae secciones de cada línea de un archivo o entrada, normalmente separadas por delimitadores (como comas o espacios).

- **`sort`**  
  Ordena líneas de texto alfabéticamente o numéricamente.

- **`uniq`**  
  Elimina o muestra líneas duplicadas en un texto ordenado.

- **`tr`**  
  Traduce o elimina caracteres.  
  Se usa para convertir mayúsculas/minúsculas o reemplazar caracteres específicos.

---

## Conceptos aplicados

- Reemplazo, eliminación e inserción de texto en archivos.  
- Procesamiento de columnas y patrones con `awk` y `sed`.  
- Filtrado y búsqueda de contenido relevante con `grep`.  
- Automatización de tareas de edición sin abrir archivos manualmente.  
- Generación de reportes y transformaciones de texto para análisis.
